FILE: backend/api/callcenter/db.py
FEATURE: Call Center Database Access Layer (Central DB Gateway)
DB: DB-1 (asyncpg / Neon)
TABLES:
  → cc_callers      — upsert_caller(): id, email, display_name, phone_number, country_code
  → cc_sessions     — create_call_log(): session_id, room_id, caller_id, department, status, created_at
                      update_call_log_status(): status, ended_at, call_duration, agent_id, wait_seconds
                      get_call_log(): SELECT by session_id OR room_id
  → agent_states    — set_agent_online/offline(): agent_id, status, department, current_call_session_id
                      get_available_agent(): SELECT WHERE status='available'
                      set_agent_busy/available/on_hold(): status transitions
  → outbound_queue  — add_to_outbound_queue(): call_log_id, user_email, department
                      complete_outbound(): UPDATE status='completed'
  → admin_config    — get_config(key): SELECT value WHERE key=?
                      set_config(key, value): UPSERT
  → transcripts     — Created in init_db() DDL; schema: id, call_id, session_id,
                        speaker, text, sentiment, created_at
TRIGGERS:
  App startup → init_db() creates all tables if missing
  Caller joins → upsert_caller() + create_call_log()
  Call status change → update_call_log_status()
  Agent login/logout → set_agent_online/offline()
  Call ends → update_call_log_status(ended/completed)
  Callback scheduled → add_to_outbound_queue()

FILE: backend/api/callcenter/api.py
FEATURE: Call Center REST API (Caller Auth, Call Initiation, Agent State)
DB: DB-1 (via db.py functions)
TABLES:
  → cc_callers      — POST /cc/auth: reads/creates caller record by email
  → cc_sessions     — POST /cc/call: creates new session, GET /cc/queue: reads sessions

                      POST /cc/outbound/complete: marks callback done
TRIGGERS:
  POST /cc/auth           → upsert_caller()
  POST /cc/call           → create_call_log() + enqueue_caller()
  POST /cc/agent/online   → set_agent_online()
  DELETE /cc/agent/offline → set_agent_offline()
  POST /cc/agent/accept   → set_agent_busy() + dequeue_caller(reason="connected")
  POST /cc/agent/end-call → dequeue_caller(reason="completed")
  POST /cc/outbound/accept → complete_outbound()
